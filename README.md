# Hydrogen template: Skeleton

Hydrogen is Shopify’s stack for headless commerce. Hydrogen is designed to dovetail with [Remix](https://remix.run/), Shopify’s full stack web framework. This template contains a **minimal setup** of components, queries and tooling to get started with Hydrogen.

[Check out Hydrogen docs](https://shopify.dev/custom-storefronts/hydrogen)
[Get familiar with Remix](https://remix.run/docs/en/v1)

## What's included

- Remix
- Hydrogen
- Oxygen
- Vite
- Shopify CLI
- ESLint
- Prettier
- GraphQL generator
- TypeScript and JavaScript flavors
- Minimal setup of components and routes

## Getting started

**Requirements:**

- Node.js version 18.0.0 or higher

```bash
npm create @shopify/hydrogen@latest
```

## Building for production

```bash
npm run build
```

## Local development

```bash
npm run dev
```

## Setup for using Customer Account API (`/account` section)

Follow step 1 and 2 of <https://shopify.dev/docs/custom-storefronts/building-with-the-customer-account-api/hydrogen#step-1-set-up-a-public-domain-for-local-development>

## Brand Guidelines

### Brand Overview
Sundae is a niche ice creamery focused on small-batch flavor experiences, warm hospitality, and a memorable neighborhood feel.

### Positioning
- **Category:** Niche ice cream shop
- **Audience:** Flavor-curious locals, families, and dessert lovers looking for premium treats
- **Promise:** Thoughtfully crafted ice cream that feels special, joyful, and approachable

### Brand Voice
Playful, warm, and confident, Sundae speaks like a friendly flavor expert who makes every scoop feel like a small celebration.

### One-Sentence Brand Voice
Sundae sounds like your cheerful neighborhood ice cream maker, inviting everyone to discover bold, handcrafted flavors with a smile.

### Typography
- **Primary Sans Serif:** Google Sans (headlines, navigation, buttons, key UI labels)
- **Secondary Serif:** Lora (supporting text, story sections, quotes, editorial accents)

### Color Palette
- **Deep Teal:** `#084C61` (brand anchors, primary backgrounds, strong contrast moments)
- **Coral Red:** `#DB504A` (calls to action, highlights, energetic accents)
- **Golden Yellow:** `#E3B505` (warmth, feature highlights, seasonal/promotional moments)

### Color Usage Balance
- **Primary base:** `#084C61` (about 60%)
- **Secondary accent:** `#DB504A` (about 25%)
- **Highlight accent:** `#E3B505` (about 15%)

### Visual Direction
- Keep layouts clean and inviting with generous spacing.
- Pair bold color moments with simple, clear typography.
- Use playful but minimal decorative elements so product imagery stays central.

### Brand Keywords
Crafted, cheerful, flavorful, welcoming, neighborhood, premium.
